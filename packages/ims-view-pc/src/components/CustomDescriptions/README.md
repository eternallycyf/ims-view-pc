## CustomDescriptions 描述列表

CustomDescriptions 是一个用于展示描述列表的组件，适用于展示多个项目的标题和相应的描述内容。

### 何时使用

- 显示一系列项目的标题和描述。
- 提供一致的描述列表样式，以保持整体设计风格。
- 允许灵活配置描述列表内容和样式。
